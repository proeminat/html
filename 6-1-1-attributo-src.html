Attributo src

Come abbiamo detto, nell’attributo src specifichiamo qual è l’origine dell’immagine. Tipicamente inseriamo una URL, o un percorso che punti ad un file.

File, percorsi relativi e assoluti

Resta valido il discorso dei percorsi relativi ed assoluti visto per i link.

Abbiamo già visto come inserire un'immagine quando il file dell’immagine si trova nella stessa cartella del file HTML, piuttosto che in una sottocartella.

Sarà quindi possibile riferirsi a file che si trovano su cartelle o siti diversi in questo modo:

Immagine che si trova nella stessa cartella del file HTML: 

<img src="miaImmagine.jpg" alt="Mia Immagine">

Immagine che si trova in una sotto-cartella del file HTML che si chiama ad esempio img: 

<img src="img/miaImmagine.jpg" alt="Mia Immagine">

Immagine che si trova su siti esterni:

<img src="https://www.iisbassi.edu.it/miaImmagine.jpg" alt="Mia Immagine">
